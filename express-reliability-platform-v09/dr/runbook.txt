Runbook: How to Fix Problems

1. See a message from AIOps or Slack? Stay calm!
2. Read what the problem is (latency, error, CPU, app fail).
3. Find the right script in the scripts/ folder to fix it.
4. If you need help, send a message to your team on Slack.
5. Practice fixing problems by running the scripts and reading this runbook.
6. If it's a big problem (Disaster!), follow the steps and ask for help.

You are now an incident hero!

# Disaster Recovery Basics

## Objectives
- RTO (Recovery Time Objective): 60 minutes
- RPO (Recovery Point Objective): 15 minutes

## Core DR Steps
1. Detect and declare incident.
2. Contain impact and protect data.
3. Recover service in secondary path or region.
4. Validate data integrity and service health.
5. Communicate recovery status.
6. Document lessons learned.

## DR Drill Cadence
- Run one tabletop drill per month.
- Run one technical failover drill per quarter.

## Evidence to Save
- Incident timeline
- Recovery commands used
- Verification screenshots or logs
- Final post-incident report

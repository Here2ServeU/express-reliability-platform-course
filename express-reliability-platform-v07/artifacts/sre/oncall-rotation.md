# On-Call Rotation Guide

## Rotation Model
- Primary on-call: first responder.
- Secondary on-call: backup responder.
- Manager on-call: escalation point.

## Shift Rules
- Weekday primary shift: 08:00 to 20:00.
- Night/weekend handoff to next assigned engineer.
- Maximum continuous primary shift: 1 week.

## Escalation Timeline
1. Minute 0: page primary.
2. Minute 10: page secondary if no acknowledgement.
3. Minute 20: escalate to manager.

## Handoff Checklist
- Open incidents
- Active mitigations
- Known risks
- Planned maintenance

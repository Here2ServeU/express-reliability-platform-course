# Express Reliability Platform V10 Capstone Notes

V10 is the final learning version before the standalone capstone. It proves the incident pipeline can
recover automatically, measure recovery time, and produce a portfolio-ready README with real evidence.

## Evidence to Capture

- Chaos suite run logs from `artifacts/evidence/`.
- Recovery actions taken by `automation/recovery_policy.sh`.
- Slack, ServiceNow, and Jira screenshots or ticket IDs.
- MTTR for pod kill, node drain, CPU pressure, and crash loop drills.

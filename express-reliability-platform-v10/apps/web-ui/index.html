The Incident Loop

One signal travels the whole platform — from detection to a one-command fix.

1 · DetectAIOps flags an anomaly against the SLO threshold.
2 · ScoreRisk score + severity + summary written as evidence JSON.
3 · AlertRich Slack alert posted, naming the resolve command.
4 · ResolveEngineer runs the named script to remediate.
5 · Self-healArgoCD reconciles the cluster back to Git state.
Every alert ends with a runnable fix, for example: ./remediation/resolve_incident.sh latency node-api

# Evidence Pack Checklist

Use this checklist to prepare for internal review, client assurance, and audit support.

## Architecture and Inventory
- [ ] Current architecture diagram
- [ ] Service inventory with owners
- [ ] Environment inventory (`shared`, `live`, etc.)

## Identity and Access
- [ ] IAM role/policy inventory
- [ ] Privileged access review evidence
- [ ] CI/CD identity federation evidence (OIDC)

## Change Management
- [ ] PR review and approval samples
- [ ] Release notes and deployment history
- [ ] Rollback record or rollback drill evidence

## Reliability Operations
- [ ] SLO/SLI catalog
- [ ] Alert configuration exports
- [ ] Incident timeline sample
- [ ] Post-incident review sample

## Data Protection and Recovery
- [ ] Encryption configuration evidence
- [ ] Backup/restore test evidence
- [ ] Data retention policy evidence

## Monitoring and Detection
- [ ] Dashboard exports (golden signals)
- [ ] Log retention and access evidence
- [ ] AIOps summary/report samples

## Governance
- [ ] Controls matrix current version
- [ ] Quarterly control review log
- [ ] Exceptions/risk acceptance register
